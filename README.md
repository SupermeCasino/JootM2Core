# mir2.core
热血传奇2核心库

针对热血传奇2的资源文件解析核心库

使用纯java代码，不依赖awt/nio等java特定平台功能，可直接用于android等后端

基于maven，可顺利使用gradle进行转换和构建

加入了一个TiledMap导出工具类，导出后效果如下(gif较大，请等待加载)
